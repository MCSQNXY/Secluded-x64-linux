## 欢迎您使用 Scluded for Linux 客户端

## 在使用之前 请您务必阅读此说明以及注意事项

## 建议使用 Ubuntu 22.04 或 Centos 9 作为 Linux 系统版本

## 打开 Linux 终端 使用命令   cd 到 SecludedLauncher.out.sh   文件 所在 目录

## 运行程序 终端输入 chmod 777 SecludedLauncher.out.sh && ./SecludedLauncher.out.sh

## 词库存放目录 ~/engine/你的机器人QQ账号/lexicon/ 里面 只识别 .txt 后缀文件

## 设备别名 请勿泄漏 谨防 设备授权 被盗！！！

## 不支持共享 设备别名 违者服务器直接踢！！！

## 常用的控制台命令

## clear 清空控制台输出

## stop 停止程序

## did+设备别名   修改设备别名

## grouplistdisable show   显示群聊禁用列表

## grouplistdisable set [群号1,群号2,以此类推]   设置群聊禁用  列如 grouplistdisable set [695014]

## friendlistdisable show  显示好友禁用列表

## friendlistdisable set    [QQ号1,QQ号2,以此类推]   设置好友禁用  列如 friendlistdisable set [695014]

## master show   显示词库主人列表

## master set [QQ号1,QQ号2,...]   设置词库主人 列如 master set [3337140142]

## wss-port 端口号      设置开放协议端口   列如 wss-port 24803

## wss-token 协议令牌       设置开放协议令牌    列如 wss-token 123456

## wss-enable 1/0            设置是否开放协议   1 开 0 关        列如 wss-enable 1








## 其他细节看   '解析 engine.json.txt'

## 您已经阅读 并 熟知

## 请把 'engine.json' 文件的 user-agent 的值 false 改成 true

## 重新启动程序 方可开始使用
